# Task Manager (Java Core Project)

This is a simple **Task Manager Simulation** built using **Java Core concepts**.

## Features
- Add a task
- Mark task as completed
- View all tasks
- Exception handling for invalid indexes

## Technologies Used
- Java Core
- Arrays
- Classes & Objects
- Exception Handling

## How to Run
1. Open a terminal in the `src` folder.
2. Compile the program:
   ```bash
   javac TaskManagerApp.java
   ```
3. Run the program:
   ```bash
   java TaskManagerApp
   ```
